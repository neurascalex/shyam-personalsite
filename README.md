# Shyam Pitla - Personal Brand Website (Next.js)

A modern, high-performance personal brand website built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion. This is a recreation of the original Vite/React project in Next.js with improved performance and SEO.

## 🚀 Features

- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Framer Motion** for smooth animations
- **Responsive Design** - Mobile-first approach
- **SEO Optimized** with proper metadata
- **Google Fonts** integration (DM Sans & Cormorant Garamond)
- **Smooth Scrolling** navigation
- **Performance Optimized** with Next.js Image component

## 📋 Sections

1. **Hero** - Introduction with key services
2. **About** - Personal story and mission
3. **Ecosystem** - Three brand overview (HR Digital Media, The Impact Show, NeuraScaleX)
4. **What I Do** - Services and expertise with testimonials
5. **Proof** - Results and success stories
6. **Advisory** - Strategic advisory services
7. **The Impact Show** - Podcast information
8. **NeuraScaleX** - AI product details
9. **CTA** - Call to action with contact options
10. **Footer** - Social links and copyright

## 🛠️ Installation

1. **Navigate to the project directory:**
   ```bash
   cd shyam-pitla-nextjs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📦 Build for Production

```bash
npm run build
npm start
```
